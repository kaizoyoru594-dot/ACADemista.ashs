Detailed Quick Commands for 2D/3D AutoCAD

  • L - Line: Draw straight lines between two points. Example: L → start point → end point.
  • C - Circle: Draw circles by specifying center and radius. Example: C → center → radius.
  • REC - Rectangle: Draw rectangles by specifying two opposite corners. Example: REC → corner1 → corner2.
  • TR - Trim: Cut unwanted parts of objects. Example: TR → select edges → trim parts.
  • EX - Extend: Extend objects to meet edges of other objects. Example: EX → boundary → extend line.
  • CO - Copy: Copy objects to a specified location. Example: CO → base point → new location.
  • MI - Mirror: Create a mirror image of objects. Example: MI → select → mirror axis.
  • AR - Array: Create multiple copies of objects in a pattern. Example: AR → select → rows/columns.
  • PL - Polyline: Draw connected lines and arcs as a single object. Example: PL → multiple segments.
  • DIM - Dimension: Add measurements to your drawing. Example: DIM → linear → points.
  • A - Arc: Draw curved lines. Example: A → start → end → radius point.
  • O - Offset: Create parallel copies. Example: O → distance → select object.
